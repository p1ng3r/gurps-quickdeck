QuickDeck Dwarven Core Batch 1 Draft

These assets are draft transparent .webp files at quality 75 for the GURPS QuickDeck dwarven UI skin.

Important:
- These were converted from generated preview assets with best-effort background alpha removal.
- Inspect edges before committing to the repo.
- Do not bake UI labels into assets; keep text as HTML/CSS.
- Use CSS fallbacks so missing assets do not create 404 noise or broken UI.
- Decorative layers must use pointer-events: none.
