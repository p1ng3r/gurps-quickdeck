# Epic Rolls PF2E

Epic Rolls PF2E is a Foundry VTT module for the Pathfinder 2E (PF2E) system.

This repository currently contains the **Phase 0 foundation** only: module metadata, startup hooks, namespace wiring, and placeholder styling.

## Purpose

The long-term goal is to provide **cinematic group roll** functionality tailored specifically to PF2E gameplay.

## Current Status

- ✅ Foundry VTT v13-compatible scaffold
- ✅ PF2E system relationship declared
- ✅ Global API namespace (`ui.EpicRollsPF2e`) with a Phase 0 placeholder `requestRoll`
- ❌ No UI yet
- ❌ No roll-resolution logic yet
