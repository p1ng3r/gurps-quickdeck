This file contains GURPS page-reference key/name mappings compatible with GCS-style page references.

The mapping list is derived from or based on Richard A. Wilkes' GCS page reference name mappings.

Source: https://github.com/richardwilkes/gcs/blob/master/ux/pageref_name_mappings.go

Original GCS file is licensed under MPL 2.0.

This data file contains only book key/name metadata, no PDFs and no copyrighted book text.

QuickDeck uses it only to help users identify which personally owned PDF to map.
