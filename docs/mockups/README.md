# QuickDeck Standalone Mockups

These files are standalone visual mockups for QuickDeck layout exploration.

## Important

- Foundry does **not** load anything in `docs/mockups/`.
- These files are safe to open directly in a local browser.
- Production migration should happen later in tiny, reviewed passes.
- Do **not** paste this mock directly into `templates/quickdeck.hbs`.

## Included mock

- `quickdeck-v0.8.1.3-command-desk.html`
- `quickdeck-v0.8.1.3-command-desk.css`

Open the HTML file directly to review the command-desk mock behavior and layout.
